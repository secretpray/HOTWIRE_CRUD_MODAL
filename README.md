<details>
  <summary>Getting Started</summary>
  
  1) Bundle install:
  
  ```ruby
  bundle install
  ```
  2) Yarn install:
 
  ```ruby
  yarn install --check-files
  ```
  3) Prepare database (db:setup or db:prepare)
 
  ```ruby
  yarn install --check-files
  ```
